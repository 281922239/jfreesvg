JFreeGraphics2D is a vector graphics library for the Java(tm) platform that allows Java2D drawings to be rendered to various output formats including Scalable Vector Graphics (SVG) and Portable Document Format (PDF).

The key classes in this library are:

  • the {@link org.jfree.graphics2d.svg.SVGGraphics2D} for generating output in Scalable Vector Graphics (SVG) format;
  • the {@link org.jfree.graphics2d.pdf.PDFGraphics2D} for generating output in Portable Document Format (PDF).

JFreeGraphics2D requires Java 6 or later.

For more information, see http://www.jfree.org/jfreegraphics2d/index.html.
