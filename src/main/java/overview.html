JFreeGraphics2D is a vector graphics library for the Java(tm) platform that allows Java2D drawings to be rendered to various output formats including Scalable Vector Graphics (SVG) and Portable Document Format (PDF). The key classes in this library are:

  • the {@link org.jfree.graphics2d.svg.SVGGraphics2D} class for generating output in Scalable Vector Graphics (SVG) format;
  • the {@link org.jfree.graphics2d.pdf.PDFDocument} and {@link org.jfree.graphics2d.pdf.PDFGraphics2D} classes for generating output in Portable Document Format (PDF).

    • In addition, there is an experimental {@link org.jfree.graphics2d.canvas.CanvasGraphics2D} class for translating Java2D calls to Javascript HTML5 Canvas API calls.

    JFreeGraphics2D requires Java 6 or later. For more information, see http://www.jfree.org/jfreegraphics2d/index.html.
